# Course Resources for 2026 NACME NVB Engineering Design

## Course Overview
In this course, you will work in teams to design and develop **data analytics apps** that support decision-making in a *smart, eco-friendly gym*. Using MATLAB, you will analyze real-world sensor data, build interactive applications, and apply engineering principles across multiple disciplines (e.g., mechanical, electrical, civil, biomedical).

This project emphasizes:
- Data-driven design and decision-making  
- App development for visualization and analysis  
- Interdisciplinary engineering thinking  
- Collaborative problem-solving  

## What’s in This Folder
This folder contains resources to support your work on the project:

### 1. Teamwork Agreement
A structured agreement to help your team:
- Define **roles and responsibilities** (e.g., data processing, modeling, app development, testing, documentation)
- Establish **communication norms and expectations**
- Plan how you will **collaborate across different aspects of the project**
- Set guidelines for accountability and conflict resolution  

You must complete and submit this agreement early in the project.

### 2. Project Assessment Rubric
A rubric that outlines how your project will be evaluated, including:
- **Technical implementation** (data analysis, modeling, app functionality)  
- **Engineering reasoning and design decisions**  
- **Quality of data visualization and user interface**  
- **Documentation and communication** (code, README, presentation/report)  

Use this rubric throughout the project to guide your work and ensure you meet expectations.

## How to Use These Resources
- Start by completing the **teamwork agreement** to organize your team.  
- Refer to the **rubric regularly** to guide your design decisions and final deliverables.  
- Treat these materials as tools to help you build a **well-documented, professional-quality project**.
