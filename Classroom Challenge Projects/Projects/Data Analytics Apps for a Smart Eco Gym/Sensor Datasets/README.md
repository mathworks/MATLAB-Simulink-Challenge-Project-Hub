# Smart Eco-Friendly Gym Synthetic Sensor Data

This folder contains synthetic sensor datasets for the Smart Eco-Friendly Gym engineering design challenge. The datasets represent one day of activity in a next-generation gym that uses sensors to monitor fitness activity, energy use, environmental conditions, air quality, structural response, and space utilization.

Each dataset is organized by **sensor type**, not by engineering discipline. Teams may choose which sensor files to use depending on the question they want to investigate and the dashboard they want to build.

The goal is to use MATLAB to turn raw or semi-processed sensor data into clear, actionable information for a non-expert audience. Students may use these data to calculate metrics, create visualizations, compare conditions, identify patterns, and design interactive MATLAB App Designer dashboards.

All data in this folder are synthetic but designed to be realistic enough for introductory engineering analysis. The values should not be interpreted as measurements from real people, real buildings, or real gym equipment.

In each of the subfolders, you'll find all the sensor data files and a detailed README describing the information that can be found in each dataset.
